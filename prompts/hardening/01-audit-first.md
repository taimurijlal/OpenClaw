# Step 1 — Audit First

**What it does:** Runs a deep security audit and explains the findings before changing anything. You read and understand before you act.

```text
Run a deep security audit and show me the findings first.

Explain:
- critical issues
- warnings
- what each means in plain English

Do not auto-fix everything blindly.
Before changing exec policy, sandbox policy, MCP settings, or gateway settings, show me the proposed changes and ask for approval.
```
