# Step 3 — Secure Messaging

**What it does:** Checks your Telegram and Discord configuration so only you can drive the agent, and flags any over-broad allowlist or public-server exposure.

```text
Check my messaging configuration for Telegram and Discord.

Please:
1. Verify only my user IDs are allowlisted
2. Enable pairing or verification for unknown DM senders where supported
3. Flag if the bot is present in public or shared servers
4. Show me the current allowlist and explain whether it is too broad

Do not change unrelated execution, sandbox, or MCP settings.
```
