# Script 8 — Security Podcast and Blog Digest

**Audience:** All security roles • **Cadence:** Weekly

Searches for the latest episodes and posts from top security podcasts and blogs. Delivers a curated reading/listening list so you stay current without spending hours browsing.

**Zapier actions to enable:**
- Slack: Send Channel Message

```text
Create my weekly security content digest.

Search the web for new episodes and posts from the
last 7 days from these sources:

PODCASTS:
- Darknet Diaries (Jack Rhysider)
- Risky Business (Patrick Gray)
- SANS Internet Stormcast
- Smashing Security (Graham Cluley)
- Malicious Life
- CyberWire Daily
- Security Now (Steve Gibson)
- Click Here (Recorded Future)

BLOGS:
- Krebs on Security
- The Hacker News
- Schneier on Security
- Troy Hunt (troyhunt.com)
- Google Project Zero
- Mandiant blog
- SentinelOne research blog
- Elastic Security Labs

For each new episode or post, include:
- Title
- Source
- 1-sentence summary of what it covers
- URL / link

Categorise as:
- BREAKING: covers active incidents or zero-days
- DEEP DIVE: technical research or analysis
- CAREER: training, certifications, industry trends
- STORY: long-form narratives, case studies

Post to Slack #security-research:
"🎧 Weekly Security Content Digest — [DATE]"
[Group by category with titles, sources, and links]
Highlight the #1 must-listen and #1 must-read at top.
```
